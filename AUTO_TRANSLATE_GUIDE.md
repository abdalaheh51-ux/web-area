# دليل الترجمة التلقائية والتصنيفات الجديدة

## 📋 نظرة عامة

تم إضافة ميزة الترجمة التلقائية من العربية إلى الإنجليزية في لوحة التحكم (قسم أعمالنا)، مع إضافة تصنيفات جديدة:
- **صفحة هبوط** (Landing Page)
- **موقع شركات** (Corporate Website)

---

## 🎯 المميزات الرئيسية

### 1. التصنيفات الجديدة
تم إضافة تصنيفان جديدان إلى قائمة التصنيفات:
- صفحة هبوط (Landing Page)
- موقع شركات (Corporate Website)

بالإضافة إلى التصنيفات الموجودة:
- تحليل بيانات (Data Analytics)
- متجر إلكتروني (E-Commerce)
- سيستم إداري (Admin System)
- بورتفوليو (Portfolio)

### 2. الترجمة التلقائية
عند كتابة النصوص بالعربية، يمكن الضغط على زر "ترجمة" لترجمة النص تلقائياً إلى الإنجليزية.

**الحقول المدعومة:**
- اسم المشروع (Project Name)
- الوصف (Description)
- المشكلة (Problem)
- الحل (Solution)
- النتيجة (Result)

### 3. إخفاء الحقول الإنجليزية
افتراضياً، يتم إخفاء حقول الإدخال الإنجليزية من الواجهة لتقليل الفوضى. يمكن إظهارها بتفعيل خيار "عرض الحقول الإنجليزية" في الإعدادات.

---

## 📁 الملفات المُنشأة

### 1. **src/app/api/translate/route.ts**
API endpoint للترجمة التلقائية
- يستخدم Google Translate API
- يدعم الترجمة من العربية إلى الإنجليزية
- معالجة أخطاء شاملة

### 2. **src/hooks/use-auto-translate.ts**
Hook مخصص للترجمة التلقائية
- إدارة حالة الترجمة
- معالجة الأخطاء
- تخزين مؤقت للنتائج

### 3. **src/app/admin/portfolio-form-enhanced.tsx**
مكون محسّن لحقول النموذج
- عرض/إخفاء الحقول الإنجليزية
- زر الترجمة التلقائية
- واجهة سهلة الاستخدام

## 🚀 خطوات التطبيق

### المرحلة 1: إضافة مفتاح Google Translate API

1. اذهب إلى [Google Cloud Console](https://console.cloud.google.com/)
2. أنشئ مشروعاً جديداً (أو استخدم مشروعاً موجوداً)
3. فعّل **Google Translate API**
4. أنشئ مفتاح API (API Key)
5. أضف المفتاح إلى ملف `.env.local`:

```env
GOOGLE_TRANSLATE_API_KEY=your_api_key_here
```

### المرحلة 2: دمج المكونات الجديدة

في ملف `src/app/admin/page.tsx`، أضف الاستيراد:

```typescript
import { EnhancedPortfolioFormFields } from './portfolio-form-enhanced'
```

ثم استبدل الحقول الحالية بـ:

```typescript
<EnhancedPortfolioFormFields
  onFieldChange={(field, value) => update(field, value)}
  formData={{
    name: formData.name,
    nameEn: formData.nameEn,
    description: formData.description,
    descriptionEn: formData.descriptionEn,
    problem: formData.problem,
    problemEn: formData.problemEn,
    solution: formData.solution,
    solutionEn: formData.solutionEn,
    result: formData.result,
    resultEn: formData.resultEn,
  }}
  dir={dir}
  hideEnglishFields={true}
/>
```

### المرحلة 3: اختبار الميزة

1. افتح لوحة التحكم
2. اذهب إلى قسم "أعمالنا"
3. أضف مشروعاً جديداً
4. اكتب الاسم بالعربية
5. انقر على زر "ترجمة"
6. تحقق من أن النص قد تم ترجمته تلقائياً

---

## 💻 مثال الاستخدام

### من لوحة التحكم:

1. افتح قسم "أعمالنا"
2. انقر على "إضافة مشروع جديد"
3. اختر التصنيف (مثلاً: "صفحة هبوط")
4. أدخل اسم المشروع بالعربية (مثلاً: "صفحة هبوط متجر إلكتروني")
5. انقر على زر "ترجمة" بجانب الحقل
6. سيتم ترجمة النص تلقائياً إلى الإنجليزية
7. كرر العملية للحقول الأخرى (الوصف، المشكلة، الحل، النتيجة)
8. احفظ المشروع

## 🐛 استكشاف الأخطاء

### المشكلة: "Translation API key not found"
**الحل:** تأكد من إضافة `GOOGLE_TRANSLATE_API_KEY` إلى ملف `.env.local`

### المشكلة: "Translation failed"
**الحل:**
1. تحقق من صحة API Key
2. تحقق من تفعيل Google Translate API في Google Cloud Console
3. تحقق من اتصالك بالإنترنت

### المشكلة: الترجمة غير دقيقة
**الحل:** Google Translate قد لا يكون مثالياً لجميع الحالات. يمكنك:
1. تعديل النص المترجم يدوياً
2. استخدام خدمة ترجمة أخرى (مثل DeepL)
