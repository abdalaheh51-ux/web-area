# دليل الاختبار الشامل - قسم Portfolio

## 🧪 خطة الاختبار

### 1. اختبار الصفحة الرئيسية - عرض الصور

#### 1.1 اختبار عرض الصور الرئيسية
- [ ] تحميل الصفحة الرئيسية بنجاح
- [ ] ظهور جميع مشاريع portfolio
- [ ] عرض الصور الرئيسية بشكل صحيح
- [ ] عند فشل تحميل الصورة، يظهر icon الفئة بدلاً من صورة مكسورة
- [ ] تأثير hover يعمل بشكل صحيح على الصور

#### 1.2 اختبار المعرض (Gallery)
- [ ] فتح تفاصيل المشروع (modal)
- [ ] عرض الصور الثلاث في المعرض
- [ ] عند فشل تحميل صورة من المعرض، يظهر icon بدلاً من صورة مكسورة
- [ ] إمكانية التمرير الأفقي في المعرض
- [ ] ظهور رسالة "اسحب لرؤية المزيد"

#### 1.3 اختبار التصفية والترقيم
- [ ] عمل أزرار التصفية (الكل، متجر، بورتفوليو، تحليل، إداري)
- [ ] تحديث الصور عند تغيير الفلتر
- [ ] عمل أزرار الترقيم (السابق/التالي)
- [ ] عرض النقاط الدالة على الصفحة الحالية

#### 1.4 اختبار الأداء
- [ ] تحميل الصور بسرعة معقولة
- [ ] عدم تأخر الصفحة عند تحميل عدد كبير من الصور
- [ ] استخدام lazy loading بشكل فعال

---

### 2. اختبار لوحة التحكم - رفع الصور

#### 2.1 اختبار رفع الصورة الرئيسية
- [ ] الضغط على زر "رفع" يفتح نافذة اختيار الملفات
- [ ] اختيار صورة يبدأ الرفع
- [ ] ظهور مؤشر التحميل أثناء الرفع
- [ ] عند نجاح الرفع، تظهر رسالة نجاح
- [ ] الصورة تظهر في المعاينة بعد الرفع

#### 2.2 اختبار رفع صور المعرض
- [ ] رفع صور المعرض الثلاث بشكل منفصل
- [ ] ظهور معاينة صغيرة لكل صورة
- [ ] عند فشل تحميل المعاينة، تظهر رسالة خطأ واضحة

#### 2.3 اختبار التحقق من الأبعاد
- [ ] رفع صورة صغيرة جداً (أقل من 400x300) يظهر خطأ
- [ ] الخطأ يوضح الحد الأدنى والأبعاد الفعلية
- [ ] رفع صورة بالأبعاد الصحيحة ينجح

#### 2.4 اختبار حد أقصى لحجم الملف
- [ ] رفع ملف أكبر من 5 ميجا يظهر خطأ
- [ ] الخطأ يوضح الحد الأقصى والحجم الفعلي

#### 2.5 اختبار صيغ الملفات
- [ ] رفع ملف JPEG ينجح ✅
- [ ] رفع ملف PNG ينجح ✅
- [ ] رفع ملف WebP ينجح ✅
- [ ] رفع ملف GIF ينجح ✅
- [ ] رفع ملف PDF يظهر خطأ ❌
- [ ] رفع ملف نصي يظهر خطأ ❌

#### 2.6 اختبار معالجة الأخطاء
- [ ] فشل الاتصال بالخادم يظهر رسالة خطأ واضحة
- [ ] فشل التخزين (Supabase/Cloudinary) يظهر رسالة خطأ مفصلة
- [ ] الخطأ لا يؤثر على الحقول الأخرى

### 4. اختبار الحالات الحدية

#### 4.1 بدون صور
- [ ] عدم وجود صورة رئيسية - يظهر icon الفئة
- [ ] عدم وجود صور معرض - يظهر icon الفئة

#### 4.2 صور مكسورة
- [ ] URL صورة غير صحيح - يظهر icon بدلاً من صورة مكسورة
- [ ] الخادم يعيد 404 - يظهر icon بدلاً من صورة مكسورة

#### 4.3 تحميل بطيء
- [ ] محاكاة اتصال بطيء - يظهر lazy loading بشكل صحيح
- [ ] الصور تحمل تدريجياً

#### 4.4 عدد كبير من الصور
- [ ] عرض 100+ مشروع - الأداء لا تزال جيدة
- [ ] التمرير بسلاسة

---

### 5. اختبار الوصول (Accessibility)

#### 5.1 لوحة المفاتيح
- [ ] التنقل بين المشاريع باستخدام Tab
- [ ] فتح المشروع بـ Enter أو Space
- [ ] إغلاق Modal بـ Escape

#### 5.2 قارئات الشاشة
- [ ] وصف واضح لكل صورة (alt text)
- [ ] وصف واضح لأزرار التحكم
- [ ] وصف واضح لرسائل الخطأ

#### 5.3 التباين والألوان
- [ ] نص واضح على الخلفية
- [ ] رسائل الخطأ لا تعتمد على اللون فقط
